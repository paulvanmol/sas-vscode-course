## Exercise 1.2 — Open Files Two Ways (3 min)

**Method A — Explorer:**
1. Click the Explorer icon in the Activity Bar
2. Expand the `programs` folder
3. Single-click `01_data_import.sas` — notice the tab title is in *italics* (preview mode)
4. Double-click the same file — the tab title is now normal (pinned open)

**Method B — Quick Open:**
1. Press `Ctrl+P`
2. Type `exploration` — the file `02_data_exploration.sas` should appear
3. Press Enter to open it

> **Question:** What is the difference between a preview tab and a pinned tab?

## Exercise 1.3 — The Command Palette (3 min)

Use the Command Palette (`Ctrl+Shift+P`) to do each of these without using the menu:

1. Open the Settings editor  
   *(hint: type `settings`)*

2. Change the color theme to something different  
   *(hint: type `theme`)*

3. Open the integrated terminal  
   *(hint: type `terminal`)*

4. Format the current document  
   *(hint: type `format`)*

> **Tip:** You do not need to type the full command name — just enough letters to find it.

## Exercise 1.5 — Split Editor (2 min)

1. Make sure `01_data_import.sas` is open
2. Right-click its tab → **Split Right**
3. In the left pane, open `02_data_exploration.sas`
4. You now have two files side by side

> **Question:** When would you use split editor in your SAS work?

Close the split by dragging the divider to one side, or close one of the tabs.

4. Click on `PROC MEANS` in the outline — VS Code jumps to that step
